Sea Land Areas Layer
====================

The sea land layer is used to distinguish between areas of water and areas of land.

+-----------+---------+------------------------+-----------------------------------------------------+
| Name      | Type    | Constraints            | Definition                                          |
+===========+=========+========================+=====================================================+
| the_geom  | POLYGON | X Y dimensions         | Geometry                                            |
+-----------+---------+------------------------+-----------------------------------------------------+
| id        | INTEGER | Primary Key            | Unique Identifier                                   |
+-----------+---------+------------------------+-----------------------------------------------------+
| type      | VARCHAR | *not null*             | Type of sea/land areas. Possible values are         |
|           |         |                        | ``land``, ``sea`` and ``water``.                    |
+-----------+---------+------------------------+-----------------------------------------------------+
